# **App Name**: Persona Portfolio

## Core Features:

- Hero Section: Display a landing page with a hero section containing a personal introduction, a profile image, and a brief overview of skills and expertise.
- Project Showcase: Implement a section to showcase selected projects with descriptions, images, and links to live demos or GitHub repositories.
- Contact Section: Integrate a contact form or a link to a contact email for potential clients or collaborators to reach out.
- Blog Integration: Include a blog or a news feed to display the user's expertise or news, making a usage of existing GO microservices.
- Resume Download: Enable downloading a resume file directly from the website.
- Resume Polisher: AI powered tool to suggest improvements or highlight key achievements for the displayed resume.

## Style Guidelines:

- Primary color: Dark teal (#008080) to evoke professionalism and creativity.
- Background color: Very dark teal (#032626), almost black, for a sophisticated and modern feel.
- Accent color: Soft light green (#90EE90) for highlights and call-to-action buttons, providing a subtle contrast.
- Headline font: 'Belleza', sans-serif, for titles and prominent text to add an artistic touch. Body font: 'Alegreya', serif, to guarantee readability.
- Use simple, line-based icons in the accent color to represent skills, technologies, and social media links. 
- Adopt a clean, single-page layout with clear sections and smooth scrolling for a seamless user experience.
- Implement subtle hover effects and transitions to add interactivity and visual interest without being distracting.
